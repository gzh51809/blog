## CSS3有哪些新的特性？

- 选择器的升级，如`tbody: nth-child(even), nth-child(odd)`
- RGBA和透明度
- 多栏布局（column-count、column-gap、column-rule、column-width）
- @Font-face, 可以用来加载字体样式
- Word-wrap、Text-overflow
- text-Shadow、Reflect
- border-radius
- Text-decoration: 对文字的更深层次的渲染
- 渐变效果
- CSS3 的盒子模型
- Transitions
- Transforms
- Animation（@keyframes）
